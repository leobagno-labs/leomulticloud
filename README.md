# LeoMultiCloud

[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Multi-Cloud Disaster Recovery Architecture project using:

- AWS (VPC, EC2, ALB, RDS, Route53)
- Azure (VNet, VM, Application Gateway)
- Terraform (Infrastructure as Code)
- Docker & Flask application
- OpenWeatherMap API integration

---

## Project Structure

- `aws/`: Terraform code for AWS infrastructure
- `azure/`: Terraform code for Azure infrastructure
- `app/`: Flask application code
- `README.md`: Project documentation
- `LICENSE`: MIT License
